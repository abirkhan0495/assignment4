# JavaScript Questions

# 1. Difference between getElementById, getElementsByClassName, querySelector

getElementById selects a single element by its unique id.
getElementsByClassName selects multiple elements that share the same class name.
querySelector returns the first element that matches a CSS selector.
querySelectorAll returns all elements that match a CSS selector.

# 2. Creating element in DOM

A new element is created using createElement().
The element is added to the page using appendChild() or append().

# 3. Event Bubbling

Event Bubbling is when an event starts on the target element and then travels upward through its parent elements.

# 4. Event Delegation

Event Delegation is attaching one event listener to a parent element to handle events for its child elements.

# 5. preventDefault vs stopPropagation

preventDefault() prevents the default behavior of an element.
stopPropagation() prevents the event from propagating to ancestor elements.
